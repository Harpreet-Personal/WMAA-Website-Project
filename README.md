## About the Project

This project focuses on the redevelopment of the WMAA website to provide a modern, secure, and user-friendly platform. The website is being designed to support key organisational needs such as information sharing, volunteer engagement, donations, and better accessibility for users.

## Repository Structure

- `frontend/` – Frontend implementation
- `backend/` – Backend logic and APIs
- `docs/` – Project documentation
- `assets/` – Images, logos, and other resources
- `client-meetings/` – Contains Minutes of Meeting notes and related records for meetings with the client
- `facilitator-meetings/` – Contains Minutes of Meeting notes and related records for meetings with the facilitator
- `deliverable-1/` – Contains documents, drafts, and supporting materials for Deliverable 1

## Technologies Used

- Frontend: HTML, CSS, JavaScript / React
- Backend: Node.js / Express / other backend framework
- Database: MySQL / MongoDB / other database
- Version Control: GitHub
